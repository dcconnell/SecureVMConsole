# SecureVMConsole

This project aims to be a quick (under 10 secs) spin-up of a VM to view/edit documents safely and effeciently via email.

Goals:
  * Support most major extensions
  * Spin-up VM in under 10 secs to view file
  * Implement a safe way to Copy-Paste info between the VM & Desktop
  * Implement a convient way to edit/stamp PDF documents
  * Implement the ability to transfer the updated document back into an Email of the users choice to send back out
  * Have a right click option when looking at an email to open the file in SecureVM (primarily for outlook)
